# AI Resume Analyzer

A Python project that compares a resume with a job description and calculates a match score using Natural Language Processing.

## Features
- Calculates similarity between resume and job description
- Uses NLP techniques for text comparison
- Outputs a resume match score

## Technologies
- Python
- scikit-learn
- Natural Language Processing

## How to Run

1. Clone the repository
git clone https://github.com/Spartan2516/ai-resume-analyzer.git

2. Go to the project folder
cd ai-resume-analyzer

3. Create virtual environment
python3 -m venv venv

4. Activate environment
source venv/bin/activate

5. Install dependencies
pip install -r requirements.txt

6. Run the program
python resume_analyzer.py
